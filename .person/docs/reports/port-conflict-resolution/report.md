# 일일 리포트: 8080 포트 점유 문제 해결

## 📅 작업 일시
2026-03-16 (월)

## 🎯 작업 목표
- Spring Boot 애플리케이션 실행 시 발생하는 'Port 8080 already in use' 오류 해결

## 🛠️ 작업 내용
1. **문제 확인:** 
   - `netstat -ano | findstr :8080` 명령을 통해 포트 8080을 점유 중인 프로세스 ID(PID) 확인
   - PID 11464가 LISTENING 상태로 점유 중임을 식별
2. **조치 사항:**
   - `taskkill /F /PID 11464` 명령으로 해당 프로세스 강제 종료
3. **결과:**
   - 포트 8080 해제 완료 및 애플리케이션 정상 실행 가능 상태 확보

## 📝 특이 사항 및 향후 과제
- Windows 환경에서 이전 실행 인스턴스가 좀비 프로세스로 남아 발생했던 것으로 추정됨.
- 향후 동일 문제 발생 시 `netstat -ano`를 통해 PID를 먼저 확인하는 루틴 권장.

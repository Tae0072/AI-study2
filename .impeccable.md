# Project Design Context (.impeccable.md)

## Design Context

### Users & Purpose
- **사용자**: 자유로운 소통을 원하는 블로그 사용자 (회원 및 비회원).
- **상황**: 텍스트 중심의 게시글과 댓글을 통해 빠르고 간결하게 정보를 공유하고 소통함.
- **목표**: 복잡한 기능 없이 "글쓰기와 읽기"라는 본질에 집중하면서도, 현대적이고 매끄러운 UX를 경험함.

### Brand Personality
- **Voice & Tone**: 신뢰감 있고 차분하지만, 상호작용 시에는 역동적이고 즐거운 반응을 보여줌.
- **3-word Personality**: 심플(Simple), 반응형(Reactive), 유려함(Fluid).
- **Emotional Goals**: 사용자가 인터페이스를 조작할 때 "도구가 살아있다"는 생동감과 만족감을 느끼게 함.

### Aesthetic Direction
- **Visual Tone**: 미니멀한 화이트 테마 기반에 강조색(Primary Blue)을 사용한 현대적 감각.
- **Theme**: 라이트 모드 중심 (다크 네이비 헤더로 무게감 부여).
- **Anti-references**: 과도하게 화려한 장식, 느린 로딩, 딱딱하고 끊기는 페이지 전환.

### Design Principles
1. **Physics-First Interaction**: 모든 버튼, 입력창, 카드는 스프링 물리 법칙(GSAP)을 따라 탄성 있게 반응해야 한다.
2. **Seamless View Transitions**: SSR 환경임에도 불구하고 View Transitions API를 적극 활용하여 앱과 같은 부드러운 화면 전환을 제공한다.
3. **Instant Feedback**: 검색이나 필터링은 사용자가 기다리지 않도록 실시간(Debounced)으로 처리하고 시각적 변화를 즉시 보여준다.
4. **Clean Typography**: 부트스트랩 기본 폰트 시스템을 존중하되, 적절한 자간과 행간을 통해 텍스트 가독성을 최우선으로 한다.
